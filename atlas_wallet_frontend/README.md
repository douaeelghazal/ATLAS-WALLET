# Welcome to your intelligent Wallet - Atlas

## To start the project run:

cd atlas_wallet_frontend
npm install
npm run dev

## Environment

Create `.env` from `.env.example` and fill values:

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_PUBLISHABLE_KEY`
- `VITE_ATLAS_API_URL` (optional)
- `VITE_ATLAS_CONTRACT_ID` (optional)

For one-service deployment of frontend + backend on Railway, see `RAILWAY_DEPLOY.md` at repository root.
